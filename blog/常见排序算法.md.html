int temp; for (int i = 0; i < length; i++) { for (int j = 0; j < length - i - 1; j++) { if (a[j] > a[j + 1]) { temp = a[j]; a[j] = a[j + 1]; a[j + 1] = temp; } } } } ``` ## 快速排序 > 选择第一个数为 p，小于 p 的数放在左边，大于 p 的数放在右边。 递归的将 p 左边和右边的数都按照第一步进行，直到不能递归。 ```java private static void quickSort(int start, int end, int[] a) { if (start < end) { int base = a[start];
