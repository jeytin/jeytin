# Compose for Desktop

## AndroidToolkit

最近学习`compose`，使用`compose for desktop`写了个玩具，相关开发工具如下：

```ini

Windows 10/11
IntelliJ IDEA 2023.1.3
Java 17
Kotlin 1.8.0
AGP 7.3.0
compose 1.3.0
Redmi Note 9
Android 12

```
